# Zoomify

*Zoomify* is a robust and flexible [WordPress](http://wordpress.org) theme, designed by [ThemeRobo](http://www.themerobo.com/) to help you that presents your work and content with absolute clarity and impact. It's available to download for free at the WordPress [theme repository](https://wordpress.org/themes/).


## Documentation
You can view detailed Zoomify documentation on the [ThemeRobo documentation web site](http://www.themerobo.com/docs/zoomify-help-file/).

## Help & Support
ThemeRobo customers can get support at the ThemeRobo support portal. Otherwise you can try posting on the [WordPress support forums](https://wordpress.org/support/theme/zoomify/).
